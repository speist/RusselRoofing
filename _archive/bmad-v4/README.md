# BMad v4 Archive

Archived on 2026-05-21 when the project upgraded to BMad v6.7.1.

These folders are no longer active and are preserved here for reference only.
The active BMad install lives in `_bmad/` (skills in `.claude/skills/`).

## Contents
- `bmad-core/` — original `.bmad-core/` v4 install
- `bmad-infrastructure-devops/` — original `.bmad-infrastructure-devops/` v4 expansion pack
- `web-bundles/` — original `web-bundles/` v4 web agent bundles
- `claude-commands-BMad/` — original `.claude/commands/BMad/` v4 slash commands
- `claude-commands-bmadInfraDevOps/` — original `.claude/commands/bmadInfraDevOps/` v4 slash commands

## Why archived not deleted
Old story files and planning artifacts may reference v4 paths. Keep until those
references are confirmed dead, then this directory can be removed.

## To restore
None of these should be restored. v6 is the active framework. If you need historical
content (e.g. an old workflow file), copy it out individually rather than reactivating
the v4 install — having both running causes skill/command name collisions.
